Swiggy Sales Analysis Dashboard (Excel)

->Overview

This project presents an interactive Excel dashboard analyzing Swiggy sales data. The dashboard provides insights into sales performance, customer behavior, and regional trends using dynamic visualizations and KPIs.

->Objectives

- Analyze overall sales performance
- Identify top-performing cities and categories
- Track monthly, weekly, and daily sales trends
- Evaluate customer ratings and order patterns


->Tools & Skills Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- KPI Design
- Dashboard Design

->Key Metrics

- Total Sales: ₹53.01M
- Average Rating: 4.34
- Average Order Value: ₹268.51
- Total Orders: 197K
- Rating Count: 5.59M

->Dashboard Preview

"Dashboard" (images/dashboard.png)

->Insights

- Bengaluru generated the highest sales among all cities
- Sales remained relatively stable with slight fluctuations across months
- Weekend sales (especially Saturday) are higher compared to weekdays
- Non-veg category contributes a larger share of revenue

->Features

- Interactive slicers (Month, Category, Restaurant)
- Sales trend analysis (Monthly, Weekly, Daily)
- City-wise performance comparison
- State-wise sales distribution map

->File Information

- "Swiggy_Sales_Analysis_Dashboard.xlsx" → Main dashboard file
- "images/" → Dashboard screenshots

->How to Use

1. Download the Excel file
2. Open in Microsoft Excel (2016 or later recommended)
3. Use slicers to interact with the dashboard
